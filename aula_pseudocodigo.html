O que é Pseudocódigo?

Pseudocódigo é uma linguagem intermediária entre o português (ou inglês) natural e uma linguagem de programação real. Ele descreve a lógica de um algoritmo de forma estruturada, sem se preocupar com a sintaxe específica de nenhuma linguagem.

Pense nisso: antes de construir uma casa, o arquiteto faz uma planta. O pseudocódigo é a planta do seu programa — você pensa na lógica primeiro, depois traduz para o código de verdade.

💡
Por que aprender? Pseudocódigo é ensinado em praticamente todos os cursos de lógica de programação (como os de vestibular de TI, Ensino Médio Técnico e primeiros semestres de faculdade). Dominar pseudocódigo significa dominar a lógica — e isso vale para qualquer linguagem.

Por que usar pseudocódigo?

1
Foco na lógica, não na sintaxe
Você não precisa lembrar de ponto e vírgula, chaves ou indentação. Concentra-se apenas em resolver o problema.
2
Universal
Um algoritmo em pseudocódigo pode ser traduzido para Python, C, Java, JavaScript — qualquer linguagem.
3
Comunicação em equipe
Desenvolvedores, analistas e até clientes conseguem entender a lógica sem saber programar.
4
Base para qualquer linguagem
Se você entende pseudocódigo, aprender Python, C ou Java fica muito mais rápido — a lógica já está na sua cabeça.

Estruturas de Controle

Condicional SE...ENTÃO...SENÃO

condicional.alg
// Estrutura básica se (nota >= 7) entao escreval("Aprovado!") senao escreval("Reprovado.") fimse // Com múltiplas condições (senão se) se (nota >= 7) entao escreval("Aprovado") senao se (nota >= 5) entao escreval("Recuperação") senao escreval("Reprovado") fimse fimse

Condicional ESCOLHA...CASO

escolha.alg
escolha (dia) caso 1: escreval("Segunda-feira") caso 2: escreval("Terça-feira") caso 3: escreval("Quarta-feira") outrocaso: escreval("Dia inválido") fimescolha

Laço ENQUANTO (while)

enquanto.alg
contador ← 1 enquanto (contador <= 10) faca escreval(contador) contador ← contador + 1 fimenquanto

Laço PARA (for)

para.alg
// Conta de 1 até 10 para i de 1 ate 10 faca escreval("Número: " + i) fimpara // Com passo personalizado (conta de 2 em 2) para i de 0 ate 20 passo 2 faca escreval(i) fimpara

Laço REPITA...ATÉ (do-while)

repita.alg
// Executa pelo menos uma vez, repete enquanto condição for falsa repita escreva("Digite um número positivo: ") leia(n) ate (n > 0)

Vetores e Matrizes

vetor.alg
var notas : vetor[1..5] de real i : inteiro soma : real inicio soma ← 0 // Ler 5 notas para i de 1 ate 5 faca escreva("Digite nota " + i + ": ") leia(notas[i]) soma ← soma + notas[i] fimpara escreval("Média: " + (soma / 5))

Fluxograma vs Pseudocódigo

Fluxograma e pseudocódigo representam a mesma lógica de formas diferentes. Abaixo veja o mesmo algoritmo (verificar se número é positivo) nas duas formas:

INÍCIO
↓
Leia N
↓
N > 0?
↓
SIM
Exibe "Positivo"
NÃO
Exibe "Não positivo"
↓
FIM
mesmo algoritmo em pseudocódigo
algoritmo "VerificaPositivo" var n : inteiro inicio escreva("Digite um número: ") leia(n) se (n > 0) entao escreval("Positivo") senao escreval("Não positivo") fimse fimalgoritmo

10 Exercícios Práticos

Clique em cada exercício para ver o enunciado e tente resolver antes de ver a solução. O aprendizado vem da prática!

01
Calculadora de Área do Retângulo
Fácil Variáveis
▼
📌 Enunciado: Leia a base e a altura de um retângulo e exiba a área. Lembre-se: Área = base × altura.
area_retangulo.alg
algoritmo "AreaRetangulo" var base, altura, area : real inicio escreva("Digite a base: ") leia(base) escreva("Digite a altura: ") leia(altura) area ← base * altura escreval("Área = " + area) fimalgoritmo
02
Verificador de Par ou Ímpar
Fácil Condicional
▼
📌 Enunciado: Leia um número inteiro e informe se ele é par ou ímpar. Dica: use o operador MOD (resto da divisão por 2).
par_impar.alg
algoritmo "ParOuImpar" var n : inteiro inicio escreva("Digite um número: ") leia(n) se (n MOD 2 = 0) entao escreval("O número é PAR") senao escreval("O número é ÍMPAR") fimse fimalgoritmo
03
Tabuada Completa
Fácil Laço PARA
▼
📌 Enunciado: Leia um número e exiba a tabuada completa desse número (de 1 a 10) usando um laço para.
tabuada.alg
algoritmo "Tabuada" var n, i : inteiro inicio escreva("Tabuada de qual número? ") leia(n) escreval("--- Tabuada de " + n + " ---") para i de 1 ate 10 faca escreval(n + " x " + i + " = " + (n * i)) fimpara fimalgoritmo
04
Média de 3 Notas com Situação
Médio Condicional
▼
📌 Enunciado: Leia 3 notas de um aluno, calcule a média e exiba a situação: Aprovado (média ≥ 7), Recuperação (média ≥ 5) ou Reprovado (média < 5).
media_notas.alg
algoritmo "MediaNotas" var n1, n2, n3, media : real inicio escreva("Nota 1: ") leia(n1) escreva("Nota 2: ") leia(n2) escreva("Nota 3: ") leia(n3) media ← (n1 + n2 + n3) / 3 escreval("Média: " + media) se (media >= 7) entao escreval("✔ Aprovado") senao se (media >= 5) entao escreval("⚠ Recuperação") senao escreval("✘ Reprovado") fimse fimse fimalgoritmo
05
Contador de Números Pares
Médio Laço ENQUANTO Condicional
▼
📌 Enunciado: Leia 10 números inteiros e conte quantos deles são pares. Ao final, exiba o total de pares encontrados.
conta_pares.alg
algoritmo "ContaPares" var n, i, contPar : inteiro inicio contPar ← 0 para i de 1 ate 10 faca escreva("Número " + i + ": ") leia(n) se (n MOD 2 = 0) entao contPar ← contPar + 1 fimse fimpara escreval("Total de pares: " + contPar) fimalgoritmo
06
Fatorial de um Número
Médio Laço PARA
▼
📌 Enunciado: Leia um número inteiro positivo N e calcule seu fatorial. Lembre-se: N! = N × (N-1) × (N-2) × ... × 1. Ex: 5! = 120.
fatorial.alg
algoritmo "Fatorial" var n, i, fat : inteiro inicio escreva("Digite N: ") leia(n) fat ← 1 para i de 1 ate n faca fat ← fat * i fimpara escreval(n + "! = " + fat) fimalgoritmo
07
Maior Valor em um Vetor
Médio Vetor Laço
▼
📌 Enunciado: Leia 7 números inteiros, armazene em um vetor e encontre e exiba o maior valor entre eles.
maior_vetor.alg
algoritmo "MaiorDoVetor" var v : vetor[1..7] de inteiro i, maior : inteiro inicio // Leitura para i de 1 ate 7 faca escreva("v[" + i + "]: ") leia(v[i]) fimpara // Encontrar maior maior ← v[1] para i de 2 ate 7 faca se (v[i] > maior) entao maior ← v[i] fimse fimpara escreval("Maior valor: " + maior) fimalgoritmo
08
Calculadora com Menu (ESCOLHA)
Médio ESCOLHA/CASO Repita
▼
📌 Enunciado: Crie uma calculadora com menu. O usuário escolhe a operação (1-Soma, 2-Sub, 3-Mult, 4-Div, 0-Sair), digita dois números e vê o resultado. Use escolha...caso e repita até o usuário escolher sair.
calculadora.alg
algoritmo "Calculadora" var op : inteiro a, b : real inicio repita escreval("1-Soma 2-Sub 3-Mult 4-Div 0-Sair") escreva("Escolha: ") leia(op) se (op <> 0) entao escreva("A: ") leia(a) escreva("B: ") leia(b) escolha (op) caso 1: escreval("Resultado: " + (a + b)) caso 2: escreval("Resultado: " + (a - b)) caso 3: escreval("Resultado: " + (a * b)) caso 4: se (b <> 0) entao escreval("Resultado: " + (a / b)) senao escreval("Erro: divisão por zero!") fimse outrocaso: escreval("Opção inválida!") fimescolha fimse ate (op = 0) escreval("Até logo!") fimalgoritmo
09
Fibonacci até N termos
Difícil Laço
▼
📌 Enunciado: Leia um número N e exiba os primeiros N termos da sequência de Fibonacci. A sequência começa: 0, 1, 1, 2, 3, 5, 8, 13, 21... onde cada termo é a soma dos dois anteriores.
fibonacci.alg
algoritmo "Fibonacci" var n, i, a, b, aux : inteiro inicio escreva("Quantos termos? ") leia(n) a ← 0 b ← 1 escreva("Fibonacci: ") para i de 1 ate n faca escreva(a + " ") aux ← a + b a ← b b ← aux fimpara fimalgoritmo
10
Ordenação Bolha (Bubble Sort)
Difícil Vetor Algoritmo Clássico
▼
📌 Enunciado: Leia 5 números, armazene em um vetor e ordene em ordem crescente usando o algoritmo Bubble Sort (ordenação por bolha). Exiba o vetor antes e depois da ordenação.
bubble_sort.alg
algoritmo "BubbleSort" var v : vetor[1..5] de inteiro i, j, aux : inteiro inicio // Leitura para i de 1 ate 5 faca escreva("v[" + i + "]: ") leia(v[i]) fimpara // Bubble Sort para i de 1 ate 4 faca para j de 1 ate (5 - i) faca se (v[j] > v[j+1]) entao aux ← v[j] v[j] ← v[j+1] v[j+1] ← aux fimse fimpara fimpara // Exibição ordenada escreva("Ordenado: ") para i de 1 ate 5 faca escreva(v[i] + " ") fimpara fimalgoritmo
