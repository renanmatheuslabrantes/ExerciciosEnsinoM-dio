# 📚 Exercícios Ensino Médio

Repositório contendo exercícios, atividades práticas e materiais desenvolvidos para alunos do Ensino Médio nas áreas de informática, lógica de programação e desenvolvimento web.

O objetivo deste projeto é disponibilizar conteúdos educacionais utilizados em aula, auxiliando estudantes no desenvolvimento do raciocínio lógico, interpretação de problemas e prática de programação.

---

## 🚀 Conteúdos Abordados

Este repositório inclui exercícios e exemplos relacionados a:

- 💻 Fundamentos de Informática
- 🖥️ Hardware
- 🌐 Redes de Computadores
- 📄 Desenvolvimento Web
- 🧠 Lógica de Programação
- 🔤 Pseudocódigo
- 🐍 Introdução ao Python
- ⚙️ Algoritmos
- 📚 Exercícios para sala de aula

---

## 🎯 Objetivo

Os exercícios foram desenvolvidos com foco em:

- Estimular o pensamento computacional;
- Melhorar a lógica de resolução de problemas;
- Preparar alunos para atividades práticas;
- Auxiliar no aprendizado de programação;
- Servir como material de apoio para estudos e aulas.

## 📖 Como Utilizar

1. Clone o repositório:

```bash
git clone https://github.com/renanmatheuslabrantes/ExerciciosEnsinoM-dio.git
```

2. Acesse a pasta do projeto:

```bash
cd ExerciciosEnsinoM-dio
```

3. Abra os exercícios no software correspondente:

- Visualg → Pseudocódigo
- VS Code → HTML, CSS, JS e Python
- Navegador → Projetos Web
